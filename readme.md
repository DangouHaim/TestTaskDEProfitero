# Test task DE Profitero

## Getting Started

### Requirements

You must have latest Ruby installed on your OS

### Dependencies
`nokogiri`,
`thread`,
`concurrent-ruby`

### Installing

#### You have to install bundler first
##### `gem install bundler`
#### Run `bundler install` to prepare gems

### Run
Use `ruby main.rb` to run
